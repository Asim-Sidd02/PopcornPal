# Popcorn Pal A Movie App UI
============

## _Popcorn Pal is a movie browsing app with a Netflix-inspired UI, offering trending movies, dynamic search, and detailed movie info. Enjoy seamless navigation and a cinematic experience powered by the TVMaze API._

## Features
------------

* **Splash Screen**:An elegant, thematic introduction to the app.
* **Home Screen**: Displays trending movies with thumbnails, titles, and summaries
* **Search Screen**: Allows users to search for movies by title and view results dynamically.
* **Details Screen**: Provides detailed information about selected movies
* **Netflix-like UI**: A sleek, dark-themed design inspired by Netflix.

## Getting Started
-------------------

To get started with the Music App, follow these steps:

### Prerequisites

* Flutter installed on your system
* A Windows or Mobile device for testing

### Installation

1. Clone the repository using `git clone https://github.com/Asim-Sidd02/PopcornPal.git`
2. Navigate to the project directory using `cd PopcornPal`
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to launch the app on your device
